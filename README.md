# Analisis Anomali Suhu Indonesia (1930-2015)

Proyek ini menganalisis pola anomali suhu di Indonesia selama periode 1930-2015 dengan menentukan penyimpangan suhu dari baseline 30 tahun pertama (1930-1959).
